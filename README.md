# CRKWH100

## Overview

CRKWH100 is a task-specific segmentation package derived from the DeepCrack dataset release.

- Task: segmentation
- Images: 100
- Annotations: 256
- Classes: 1 (crack)
- Annotation format: paired bitmap masks

The dataset contains line-array pavement images captured under visible-light illumination.

- Name: CRKWH100
